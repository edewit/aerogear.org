Setting up AeroGear Core

CLIENT

1. Get your environment ready

Depending on which platforms you target, you have different pre-requisites.

  • Java
  • iOS
  • JavaScript
  • Cordova

You need Android Maven SDK Deployer installed and configured.
Read more on Get started with android | Eclipse | Intellij | Netbeans

                    

You need XCode IDE installed on your machine. If not, you can download it from the Apple developer web site.
Read more on Set up AeroGear and Xcode | AeroGear Xcode Template

                    

You need an HTML5 compatible browser is required such as Chrome, Safari 5+, Firefox 5+, or IE 9+.
Read more on Set up AeroGear and JS
                    

You need to install CordovaLib, Cordova Framework and Cordova XCode templates. Download Apache Cordova 2.0.0 and unzip.
Read more on Setting up | Going Hybrid

2. Get AeroGear

  • Java
  • iOS
  • JavaScript
  • Cordova

If you use maven

  org.jboss.aerogear
  aerogear-android
  1.3.1
  apklib

If you use gradle (as experimental)
dependencies {
  compile 'org.jboss.aerogear:aerogear-android:1.3.1'
}


If you use CocoaPods in your Podfile:
xcodeproj 'YourProject.xcodeproj'
platform :ios, '7.0'
pod 'AeroGear', '1.4.0'
